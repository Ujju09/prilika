The exported JSON contains comprehensive agent behavior data following ML evaluation best practices:

  • Input Context: Original source text, transaction type, and reference
  • Maker Output: AI reasoning, confidence scores, warnings, and generated journal entries
  • Checker Output: Validation status, errors, warnings, and summary
  • Ground Truth: Final human review decisions and notes
  • Performance Metrics: Token usage and processing duration
  • Complete Logs: Full prompts and completions from each agent stage

Use this data to analyze agent performance, identify failure patterns, create test datasets, and improve skill definitions through fine-tuning or prompt engineering.
